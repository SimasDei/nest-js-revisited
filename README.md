# Nest JS Revisited

## Description
This project is a revisited version of a NestJS application. NestJS is a progressive Node.js framework for building efficient, reliable, and scalable server-side applications.

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/your-repo/nestjs-revisited.git
    cd nestjs-revisited
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

## Running the app
1. Development:
    ```bash
    npm run start
    ```

2. Watch mode:
    ```bash
    npm run start:dev
    ```

3. Production mode:
    ```bash
    npm run start:prod
    ```

## Test
1. Unit tests:
    ```bash
    npm run test
    ```

2. End-to-end tests:
    ```bash
    npm run test:e2e
    ```

3. Test coverage:
    ```bash
    npm run test:cov
    ```
